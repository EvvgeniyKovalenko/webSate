Отделение сбора технологического конденсата

Технологический конденсат из аппаратов 84D01, 84D02, 78D03, 16D02, 12E08, 12T03, насосов 12Р02А/В, 12Р05А/В поступает в коллектор сбора технологического конденсата и далее в емкость сбора технологического конденсата 97D01. Уровень (20-87,7 %) в емкости 97D01 регулируется контуром поз. 97LICA0003A/B, с сигнализацией на АРМ максимального 87,7 % и минимального 20 % значений, корректирующий сигнал которого поступает на прибор поз. 97FIC0008, клапан-регулятор которого 97FV0008 установлен на выкиде насосов 97P01A/B. При минимальном блокировочном значении уровня 17 % по сигналу одного датчика из двух контура поз. 97LICA0003A/B срабатывает сигнализация на АРМ, происходит останов насосов 97P01A/B. По месту уровень в углеводородной части емкости 97D01 контролируется прибором поз. 97LG0020. Уровень в водяной части емкости 97D01 контролируется прибором 97LG0002. Давление (0,04 МПа) в емкости 97D01 регулируются контурами поз. 97PICA0002 с сигнализацией на АРМ по верхнему пределу 0,28 МПа, клапан-регулятор 97PV0002 которого установлен на трубопроводе сброса в коллектор для газовых сбросов теплой факельной системы и поз. 97PIC0311, клапан-регулятор которого 97PV0311 установлен на трубопроводе подачи азота низкой чистоты в емкость 97D01. По месту давление в емкости контролируется прибором поз. 97PG0001.

Для защиты емкости 97D01 от превышения давления установлены СППК поз. 97SV001A/B, со сбросом в коллектор для газовых сбросов теплой факельной системы, с установочным давлением 0,4 МПа. Температура в емкости 97D01 контролируется прибором 97TG0007. Технологический конденсат из емкости 97D01 отводится (передавливанием) в сепаратор 90D01. Технологический конденсат из емкости 97D01 поступает на прием насоса откачки технологического конденсата 97P01A/B. На приеме насосов 97P01A/B установлены фильтры XAF058 и XAF059 соответственно. Перепад давления в фильтрах XAF058, XAF059 контролируется приборами поз. 97PDG0103 и 97PDG0203 соответственно. Давление на приеме насосов 97P01A/B контролируется по месту приборами поз. 97PG0101 и 97PG0201. Температура подшипников насоса 97P01A контролируется контурами поз. 97TISA0814 и 97TISA0816 с сигнализацией на АРМ максимального значения 100 °С, при достижении верхнего блокировочного значения 110 °С срабатывает сигнализация на АРМ, происходит останов насоса 97P01A.

Температура подшипников насоса 97P01В контролируется контурами поз. 97TISA0815 и 97TISA0817 c сигнализацией на АРМ максимального значения 100 °С, при достижении верхнего блокировочного значения 110 °С срабатывает сигнализация на АРМ, происходит останов насоса 97P01В. Наличие жидкости в бачке торцевого уплотнения насосов 97P01A/B контролируется контурами поз. 97LSA0820 и 97LSA0821 соответственно, при отсутствии жидкости срабатывает сигнализация на АРМ, происходит останов насосов. На насосах 97Р0А/В не предусмотрена система пуска резервного насоса при остановке рабочего. Давление в системе торцевого уплотнения насоса 97P01А контролируется контуром 97PIA0818 с сигнализацией на АРМ по верхнему пределу 0,15 МПа. По месту контроль давления и температуры осуществляется приборами поз. 97PG0810 и 97TG0812 соответственно. Давление в системе торцевого уплотнения насоса 97P01В контролируется контуром 97PIA0819 с сигнализацией на АРМ по верхнему пределу 0,15 МПа. По месту контроль давления и температуры осуществляется приборами поз. 97PG0811 и 97TG0813 соответственно. Давление на выкиде насосов 97P01A/B контролируется приборами поз. 97PG0102 и 97PG0202 по месту.

Согласно технологической схеме смонтирован трубопровод минимального протока с нагнетательного трубопровода насосов 97P01A/B обратно в емкость 97D01. С нагнетательного трубопровода насосов 97P01A/B технологический конденсат откачивается в термический окислитель 12Y03. Расход технологического конденсата регулируется контуром поз. 97FIC0008, с коррекцией по уровню в емкости 97D01 контуром 97LICA0003A/B, клапан-регулятор 97FV0008 которого установлен на нагнетательном трубопроводе насосов 97P01A/B. При отключении термического окислителя для окисления технологического конденсата 12Y03 в аварийной ситуации предусматривается возможность вывода технологического конденсата из емкости сбора технологического конденсата 97D01 на площадку очистных сооружений в резервуар производственных сточных вод 220D10 с последующим вывозом из него автотранспортом в специализированную организацию. Дождевые сточные воды с площадки налива СПГ в автотранспорт откачиваются из приямка 97D03 насосом для откачки из накопительного приямка (по ГП тит.15.2) в систему производственно-дождевой канализации.

Температура среды на приеме насоса 97P03 контролируется контуром поз. 97TZIA0002А/В с сигнализацией на АРМ по нижнему пределу 3 °С, при достижении минимального блокировочного значения 1 °С по сигналу одного датчика из двух прибора поз. 97TZIA0002А/В срабатывает сигнализация на АРМ, происходит останов насоса 97Р03. Давление на выкиде насоса 97Р03 контролируется по месту прибором поз. 97PG0006. Уровень (20-80 %) в приямке поз. по ГП тит.15.2 контролируется контуром поз. 97LISA0004, с сигнализацией на АРМ по верхнему значению 80 %, при достижении нижнего блокировочного значения уровня 20 % происходит останов насоса 97Р03. Дождевые сточные воды с площадки резервуара СПГ 73D01 откачиваются из приямка поз. по ГП тит.15.1 насосом для откачки из накопительного приямка 97P02 в систему производственно-дождевой канализации. Температура среды на приеме насоса 97P02 контролируется контуром поз. 97TZIA0001А/В с сигнализацией на АРМ по нижнему пределу 3 °С, при достижении минимального блокировочного значения 1 °С по сигналу одного датчика из двух прибора поз. 97TZIA0001А/В срабатывает сигнализация на АРМ, происходит останов насоса 97Р02. Давление на выкиде насоса 97Р02 контролируется по месту прибором поз. 97PG0019. Уровень (20-80 %) в приямке 97D02 контролируется контуром поз. 97LISA0017, с сигнализацией на АРМ по верхнему 80 % пределу, при достижении нижнего блокировочного значения уровня 20 % происходит останов насоса 97Р02.
